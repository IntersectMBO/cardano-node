Calls cardano-cli to build a simple ADA transfer transaction to the specified outputs of the specified amount of ADA. Destination address may be specified as a PaymentKeyInfo or with a script file. For each output, an extra optional script file may be provided, and if provided, the script provided will be published in that output as a reference script.
