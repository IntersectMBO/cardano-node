-> m (Maybe (String, Word16)) 

This function obtains the identifier for the last enacted parameter update proposal if any.

If no previous proposal was enacted, the function returns Nothing. If there was a previous enacted proposal, the function returns a tuple with its transaction identifier (as a String) and the action index (as a Word16).
