type AcceptedMetrics = TVar (Map NodeId MetricsStores) Source #

We have to create EKG.Store and MetricsLocalStore to keep all the metrics accepted from the node.

type ConnectedNodes = TVar (Set NodeId) Source #

This set contains ids of currently connected nodes. When the node is connected, its id will be added to this set, and when it is disconnected, its id will be deleted from this set. So, ConnectedNodes is used as a "source of truth" about currently connected nodes.

type ConnectedNodesNames = TVar (Bimap NodeId NodeName) Source #
