Cardano.Tracer.Handlers.ReForwarder

Description

This module initializes a reforwarding service for use by cardano-tracer. It could [re-] serve the three miniprotocols on a new local socket. Currently, - it reforwards trace messages to the new socket, optionally filtering trace messages. - it does not (currently) reforward EKG to the new socket. - it creates a new Datapoint store, but the datapoint store is empty.
