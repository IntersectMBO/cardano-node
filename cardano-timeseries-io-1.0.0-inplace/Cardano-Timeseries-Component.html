modifyConfig :: TimeseriesHandle -> (TimeseriesConfig -> Maybe TimeseriesConfig) -> IO () Source #

Reconfigure the store. The new parameters are applied immediately. Wakes up the pruner thread as well. If the supplied config is empty, reconfigures back to the default config.

prune :: TimeseriesHandle -> IO () Source #

Prune the store of the entries past the configured retention period (relative to "now"). The pruner thread uses that function internally, but one can still invoke the pruning manually at will.
