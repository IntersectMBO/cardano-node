runAcceptors :: TracerEnv -> TracerEnvRTView -> IO () Source #

Run acceptors for all supported protocols.

There are two "network modes" for acceptors: 1. Server mode, when the tracer accepts connections from any number of nodes. 2. Client mode, when the tracer initiates connections to specified number of nodes.
