runPrometheusServer :: TracerEnv -> Endpoint -> IO RouteDictionary -> IO () Source #

Runs a simple HTTP server that listens on endpoint.

At the root, it lists the connected nodes, either as HTML or JSON, depending on the requests 'Accept: ' header.

Routing is dynamic, depending on the connected nodes. A valid URL is derived from the nodeName configured for the connecting node. E.g. a node name of `127.0.0.1:30004` will result in the route `/12700130004` which renders that node's Prometheus / OpenMetrics text exposition:
