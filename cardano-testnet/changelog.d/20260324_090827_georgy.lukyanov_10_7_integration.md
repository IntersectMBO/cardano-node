### Added

- Support for running with KES agent via `--use-kes-agent` flag, allowing
  testing block production with kes-agent.

### Changed

- Adapted to `NetworkTopology` type changes: `createTestnetEnv` now creates a
  concrete topology and allocates random ports to nodes, instead of relying on
  an abstract topology.
- Adapted to changes in `TxSubmitResult`.
- Adapted to latest ledger changes (blockfrost response types to match alonzo
  genesis ones).
- Bumped dependencies: `ouroboros-network`, `ouroboros-consensus`,
  `cardano-api-10.25.*`, `cardano-crypto-class-2.3.*`.
